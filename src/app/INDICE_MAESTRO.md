## 🎓 GLOSARIO DE TÉRMINOS

Para referencia en todos los documentos:

- **Figma Make**: Entorno de desarrollo donde creas el código
- **Vercel**: Plataforma de hosting donde se publica la app
- **Git**: Sistema de control de versiones
- **GitHub/GitLab**: Plataformas para alojar repositorios Git
- **Repositorio**: Carpeta con tu código versionado
- **Commit**: Guardar cambios con una descripción
- **Push**: Subir commits a GitHub/GitLab
- **Pull**: Descargar cambios de GitHub/GitLab
- **Deploy**: Publicar la aplicación en producción
- **Build**: Compilar el código para producción
- **Variable de entorno**: Configuración secreta (API keys, etc.)
- **Branch**: Rama de desarrollo paralela
- **Main/Master**: Rama principal del repositorio

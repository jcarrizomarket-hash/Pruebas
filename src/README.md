# 🎯 Gestión de Servicios - Sistema de Camareros

Sistema completo de gestión de camareros integrado con Supabase, con manejo de eventos, turnos, estados de comunicación visuales, y funcionalidades de WhatsApp/Email.

---

## 🚀 INICIO RÁPIDO

### ¿Primera vez aquí?

**→ Lee [START-HERE.md](./START-HERE.md)** - Guía visual paso a paso (7-8 minutos)

**→ O lee [QUICK-START.md](./QUICK-START.md)** - Versión texto (5 minutos)

### ¿Ya sabes lo que haces?

```bash
# Instalación y deployment en un solo comando
npm install -g supabase vercel
npm install
cp .env.example .env
# Edita .env con tus credenciales
./deploy.sh
```
